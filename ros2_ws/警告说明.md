警告有两种：电机状态警告和逆运动学警告。

# 逆运动学警告

期望的某端位姿，超出工作空间，逆运动学计算失败而发出的警告：

```cpp
//light_lift_arm_6dof.cpp 中 comput_ik 函数
    if (arm_lenght < (desir_arm_lenght - 0.01))
    {
        std::cerr << "期望轨迹超出工作空间: " << std::endl;
        std::cerr << "arm_lenght:  "<< arm_lenght << std::endl;
        std::cerr << "desir_arm_lenght:  "<< desir_arm_lenght << std::endl;
        return;
    }
```

奇异位置导致运动学计算失败而发出的警告：

```cpp
//light_lift_arm_6dof.cpp 中 comput_ik 函数
    int ret = ik_solver_lma->CartToJnt(current_joint_positions, desir_end_effector_frame, desir_joint_positions_temp);

    if (ret != 0)
    {
        std::cerr << "ik_solver_lma->JntToJacDot 计算失败, 错误码 = " << ret << std::endl;
        return;
    }
```

运动学计算出的关节角度超出关节限制而发出的警告：
```cpp
//light_lift_arm_6dof.cpp 中 comput_ik 函数
    if (desir_joint_positions_temp(i) > desir_joint_pos_protect_max[i])
            {
                std::cerr << "关节 " << i << " 角度max超出工作空间" << std::endl;
                return;
            }
```


# 电机状态警告

期望的电机状态（包括位置，速度，力矩），超出限制而发生的警告：

```cpp
//motor_control.cpp 中 ControlMotors_g 函数
std::cout << "joint" << i << " desir pos out range:" << " " << pos[i]<<std::endl;

std::cout << "joint" << i << " desir vel out range:" << " " << vel[i] << std::endl;

std::cout << "joint" << i << " desir tau out range:" << " " << tau[i] << std::endl；    
```

当前的电机状态（包括位置，速度，力矩），超出限制而发生的警告：
 
```cpp
//motor_control.cpp 中 real_time_motor_protection 函数
std::cout << "joint pos " << i << " out range must Ctr + c kill programe!!!!" << pos_[i]<< std::endl;

std::cout << "joint vel " << i << " out range must Ctr + c kill programe!!!!" << vel_[i]<< std::endl;

std::cout << "joint tau " << i << " dengerous must Ctr + c kill programe!!!!"<< tau_[i] << std::endl;
  
```
这种警告会直接失能机械臂！！
